VIP Cipher Terminal
===================

How to use:
1. Open index.html in your browser.
2. Type a codename and press Authorize.
3. Send messages into the local VIP terminal.
4. Export logs if you want a JSON copy.
5. Purge clears saved browser messages.

Notes:
- Frontend only.
- Messages are stored in your browser using localStorage.
- No backend or online database is included.
